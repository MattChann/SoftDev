This is a Jumbotron (like the one above)


It's basically a large box that you can put stuff in. It's a nice way to isolate and call attention to something (ex: welcoming users of a website). It is not limited to a blue box, it can be anything from a color to an image to a video. To create a jumbotron you use a <div> element with class jumbotron. With Bootstrap you can also include multiple classes separated by spaces like adding the class text-center to this box.

P.S. You can add code elements like the ones in the last line with the <code> tag.


Navigation Bar

That dark gray bar that always sticks to the top of your screen is this is the navigation bar. We made it so that it is always there on your screen but that's just one of many customization options. You can change the colors, the links, the brand, the logo, etc. You can even add a dropdown menu to it or a form like a search bar.
